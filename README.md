# 🔒 Шифрователь файлов (AES-256)

Простой и безопасный веб-сервис для шифрования любых файлов. 

**Главная фишка:** файлы и ключи обрабатываются только в оперативной памяти сервера (ОЗУ) и **никогда не сохраняются на диск**. Полная конфиденциальность (Zero-Knowledge).

---

## 🛠 Технологии
* **Бэкенд:** Python 3 + Flask
* **Криптография:** PyCryptodome (алгоритм AES-256-CBC)
* **Фронтенд:** HTML5, CSS3, Vanilla JS (без тяжелых фреймворков)

---

## 📂 Структура проекта
* `crypt.py` — крипто-ядро (шифрование, дешифровка, генерация ключей).
* `app.py` — веб-сервер и обработка запросов (API).
* `templates/index.html` — внешний вид приложения и логика в браузере.

---

## 🚀 Быстрый запуск

1. Установи зависимости:
   ```bash
   pip install flask pycryptodome

    Запусти сервер:
    Bash

    python app.py

    Открой в браузере: http://127.0.0.1:5000

📖 Как пользоваться
1. Шифрование

    Нажми «Сгенерировать» (появится ключ из 64 символов).

    Обязательно сохрани этот ключ! Без него файл не вернуть.

    Перетащи любой файл в окно. Кнопка сама изменится на «Зашифровать».

    Нажми кнопку — скачается защищенный файл .enc.

2. Расшифровка

    Вставь свой сохраненный ключ.

    Перетащи файл .enc. Кнопка сама изменится на «Расшифровать».

    Нажми кнопку — скачается твой исходный файл.

🛡 Безопасность и ограничения

    Лимит: файлы строго до 1 ГБ (защита сервера от перегрузки).

    Ключ: должен быть ровно 64 символа (0-9, a-f).

    Защита: для каждого файла создается случайный вектор (IV), что делает шифр уникальным.
