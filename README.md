Markdown# 🔒 Шифрователь файлов (AES-256-CBC)

Легковесное и безопасное веб-приложение на Flask для шифрования и расшифровки файлов любых форматов в один клик.

Приложение разработано по принципу **Zero-Knowledge**: секретные ключи и пользовательские файлы никогда не сохраняются на диске сервера. Все криптографические операции происходят исключительно в оперативной памяти (ОЗУ) «на лету», что гарантирует абсолютную конфиденциальность.

---

## 🚀 Функциональные возможности

* **Генерация ключей:** мгновенное создание случайного 256-битного криптографического ключа в безопасном HEX-формате (64 символа).
* **Шифрование:** преобразование любого файла в защищенный бинарный формат с расширением `.enc`.
* **Расшифровка:** гарантированное восстановление исходного файла из формата `.enc` при наличии корректного ключа.
* **Умный интерфейс (Dynamic UI):** автоматическое определение типа операции. Кнопка сама переключается между «Зашифровать» и «Расшифровать» в зависимости от расширения выбранного файла.
* **Удобство использования:** поддержка технологии Drag-and-Drop для быстрой загрузки файлов через простое перетаскивание.
* **Безопасная очистка:** автоматическое аннулирование временных ссылок в браузере (Blob URL) сразу после скачивания файла для защиты ОЗУ клиента.

---

## 🛠 Технологический стек

* **Бэкенд (Backend):** Python 3, Flask (микрофреймворк для реализации REST API).
* **Криптография:** Библиотека `PyCryptodome` (модули `AES`, `Random`, `Padding`).
* **Фронтенд (Frontend):** HTML5, адаптивный CSS3, Vanilla JavaScript (асинхронные запросы через Fetch API, FormData, Blob).
* **Архитектура:** Клиент-серверная (асинхронное REST API взаимодействия).
